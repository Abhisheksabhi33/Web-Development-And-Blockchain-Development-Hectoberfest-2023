toastKing.js

toastKing is a light-weight JavaScript library for creating non-blocking, beautiful notification messages.

Demo
Star this project

    toastKing({                  
        message:'hello globe!',     
        duration:5,
        progressBarVis:true,
        borderVis:false,
        shadowVis:true,
        type:'success',
        design:'plain',
        stopOnFocus:true,
        onClick:function(){},
        onDismiss:function(){}
    })
